# OpenClaw Skills Pack

This repo contains reusable OpenClaw skills:

- `advanced-calendar` (from openclaw/skills archive)
- `news-aggregator-skill` (from openclaw/skills archive)
- `finance-analyst`
- `notebooklm-deck`

## Install

Clone this repo, then copy the skill folders into your OpenClaw skills directory, or point your agent to this repo and load the desired skill folder directly.

Example:

```bash
git clone <repo-url>
./install.sh ~/.openclaw/skills
```

## Notes

- No secrets are stored here.
- Each skill is self-contained.
- Keep your agent configuration, tokens, and personal memory outside this repo.
