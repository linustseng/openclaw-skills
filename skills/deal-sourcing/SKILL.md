# Deal Sourcing

Source targets systematically. Start from the investment mandate, build the target archetype, identify candidate companies, and rank them by fit and likelihood of actionable outreach.

## Workflow

### 1. Define the sourcing mandate

Clarify or infer:
- sector / subsector
- geography
- revenue / EBITDA range
- ownership profile: founder-owned, family-owned, sponsor-backed, carve-out, etc.
- growth, margin, and business quality preferences
- whether the goal is platform, add-on, minority, or broader pipeline building

If the user gives only a broad starting point, state the assumptions used instead of asking a long questionnaire.

### 2. Build the target archetype

Describe what a good target looks like.

Include:
- what the company does
- size and stage
- ownership pattern
- reasons it may be sellable or approachable
- strategic fit with the mandate
- common red flags

This is the sourcing lens that explains why names make the list.

### 3. Generate the candidate universe

Create a shortlist of companies or target archetypes using the mandate.

For each candidate include:
- company name
- geography
- what it does in one line
- likely fit with the mandate
- estimated size band if known
- likely ownership / seller context if inferable
- why it may be worth approaching or screening next

If the user wants a structured sourcing memo or pipeline list, read `references/output-template.md` before drafting.

### 4. Rank and prioritize

Do not stop at a raw list. Rank candidates by:
- fit with mandate
- likely actionability
- distinctiveness / strategic attractiveness
- probability the company is actually sourceable
- amount of follow-up work needed

Use labels such as `high priority`, `medium priority`, `watch`, or `low probability`.

### 5. Recommend next actions

End with concrete next steps, for example:
- screen first
- gather more facts
- look for ownership context
- warm intro / outreach
- park for later monitoring

## Missing-data behavior

If the sourcing mandate is incomplete, do not stop at asking for more. First produce a sourcing frame with explicit assumptions, then list the minimum missing inputs that would improve the list.

## Output standard

Default to a concise sourcing memo with:
1. Mandate summary
2. Target archetype
3. Prioritized candidate list
4. Why each name matters
5. Suggested next steps

## Quality bar

- Source against a real mandate, not vague theme-chasing.
- Prefer 5-10 good names over 30 weak names.
- Distinguish a great business from a realistically sourceable deal.
- Keep unknown ownership or size assumptions explicit.
- Do not confuse sourcing with full diligence; this is the top of the funnel.
