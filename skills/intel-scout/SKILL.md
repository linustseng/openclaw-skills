# Intel Scout

Scout the web for relevant developments, then summarize the useful signal and send it to the right destination.

## Workflow

1. Define the topic, company, sector, geography, and time window.
2. Search broadly, then narrow to the highest-signal sources.
3. Prefer recent coverage and primary sources when available.
4. Summarize what changed, why it matters, and what to watch next.
5. If the user wants delivery, format for Telegram first, email second.

## Notes

- Use web search plus fetch-style extraction when you need source text.
- Keep summaries short, factual, and action-oriented.
- If sources conflict, say so plainly.
