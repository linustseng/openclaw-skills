# Calendar Guardian

Use this skill to read calendars, find free slots, create events, and coordinate schedules with Google Workspace.

## Workflow

1. Clarify timezone, attendees, date range, and whether work + personal calendars should be combined.
2. Check availability before proposing times.
3. Prefer concise Chinese replies and include the next actionable slot.
4. If creating events, confirm title, time, duration, and guests before writing.
5. For follow-ups, summarize the chosen slot and any conflicts.

## Notes

- Default to combining work + personal calendars when searching.
- Use Gog for Calendar, and Gmail/Drive only when needed for supporting context.
- Keep scheduling decisions explicit, especially when multiple people are involved.
