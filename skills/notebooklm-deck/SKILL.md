# NotebookLM Deck

Prepare clean source briefs and slide prompts for NotebookLM, with strong structure and low-text decks.

## Workflow

1. Clarify audience, topic, language, length, and desired tone.
2. Distill the source into a short, clean brief before asking for slides.
3. Prefer one idea per slide, strong section breaks, and sparse text.
4. Ask for speaker notes when the presenter needs guidance.
5. If style matters, specify the visual direction explicitly.

## Style defaults

- Executive/business deck: premium, clean, confident.
- Teaching deck: clear, structured, not crowded.
- Keep prompts concrete and avoid generic slide filler.
