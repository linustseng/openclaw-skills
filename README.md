## Install

Clone this repo, then copy the skill folders into your OpenClaw skills directory.

```bash
git clone <repo-url>
cd openclaw-skills
./install.sh ~/.openclaw/skills
```

## Notes

- No secrets are stored here.
- Keep your agent configuration, tokens, and personal memory outside this repo.
- Some upstream skills include their own docs/scripts/assets as-is.
