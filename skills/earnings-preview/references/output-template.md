## Style rules
- Keep it to one page unless the user asks for more depth.
- Always preserve the section structure, even when some fields are `unknown`.
- Use source names and dates where possible.
- Focus on 3-5 important metrics, not everything available.
- Distinguish a fundamental beat from a stock beat.
- Avoid fake precision when the setup is noisy.
- Never output raw working notes or keyword dumps.
